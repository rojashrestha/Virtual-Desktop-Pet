# 🐱 Momo Cat - Virtual Desktop Pet

Momo Cat is a fun virtual desktop pet built with Python, Tkinter, and Pillow. The cat lives on your desktop, moves around, reacts to interactions, and maintains its own happiness and hunger levels.

## Features

* Interactive desktop pet
* Drag and move anywhere on screen
* Feed, play, and pet the cat
* Sleep mode
* Random wandering animation
* Speech bubbles and reactions
* Happiness and hunger system
* Custom sprite support
* Automatic fallback cat drawing

## Technologies Used

* Python
* Tkinter
* Pillow (PIL)

## Installation

```bash
pip install pillow
```

## Run

```bash
python momo_cat.py
```

## Controls

* Left Click + Drag → Move the pet
* Double Click → Pet the cat
* Right Click → Open action menu

## Available Actions

* Pet
* Feed
* Play
* Sleep
